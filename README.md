# Daily Reporter

업무 시간을 기록하고 일일/주간 보고서를 자동 생성하는 Windows 데스크탑 앱입니다.

---

## 기능

- **출근/퇴근 기록** — 클릭 한 번으로 타임스탬프와 함께 로그 파일에 저장
- **실시간 시계** — 현재 시각을 메인 화면에 표시
- **주기 알림** — 설정한 주기(기본 60분)마다 알림음과 함께 업무 내용 입력 팝업 표시 (`Enter`로 제출)
- **마지막 기록 표시** — 가장 최근 기록을 메인 화면에 표시, 파일 외부 수정 시 자동 반영
- **복사 버튼** — 마지막 기록의 업무내용을 클립보드에 복사
- **로그 확인** — 오늘 로그 파일을 기본 앱으로 바로 열기
- **일일보고 생성** — 내장 템플릿 기반으로 당일 로그를 채운 hwpx 파일 자동 생성
- **주간보고 생성** — 내장 템플릿 기반으로 기간 내 로그를 집계한 hwpx 파일 자동 생성
- **출근 상태 자동 복원** — 앱 재시작 시 오늘 로그에 출근 기록이 있으면 알람 자동 재개
- **자동 시작** — 윈도우 부팅 시 자동 실행 설정/해제 (설정 메뉴)

**설정 메뉴 항목:**

| 항목 | 설명 |
|------|------|
| 윈도우 시작 시 자동 실행 | 부팅 시 앱 자동 실행 (레지스트리 등록) |
| 이전 내용 자동 입력 | 팝업 열릴 때 직전 PROJECT명·업무내용 자동 채움 |
| 시작 시 출근 상태 자동 복원 | 재시작 시 오늘 로그에 출근 기록이 있으면 알람 재개 |
| 알림 주기 설정... | 알림 간격을 분 단위로 설정 (기본 60분) |
| 보고자 성명 설정... | 보고서에 삽입될 이름 (기본 홍길동) |

## 다운로드 및 실행

1. [dist/daily_reporter.exe](dist/daily_reporter.exe) 다운로드 후 원하는 폴더에 저장 (단일 파일, 별도 설치 불필요)
2. `daily_reporter.exe` 실행
3. 최초 실행 시 **설정 → 보고자 성명 설정...** 에서 이름 입력
4. **출근** 버튼 클릭 → 출근 시각 기록 + 정시 알림 시작
5. 매 정시마다 팝업이 뜨면 PROJECT명·업무내용 입력 후 **기록** 또는 `Enter`
6. **퇴근** 버튼 클릭 → 퇴근 시각 기록 + 알림 종료
7. **일일보고 생성** 버튼 → 당일 보고서 hwpx 파일 자동 생성 및 열기

> 로그 파일과 생성된 보고서는 `daily_reporter.exe`가 위치한 폴더에 저장됩니다.
> 템플릿 파일은 EXE에 내장되어 있어 별도 파일이 필요 없습니다.

## 로그 파일 형식

날짜별로 `YYYY-MM-DD.csv` 파일로 저장됩니다. 4개 열(date, time, project, description).

```
2026-05-12,09:03:52,출근,
2026-05-12,10:00:00,피플카운터,ESP32 BLE 스캔 최적화
2026-05-12,11:00:00,"피플카운터, 집합수요",파라미터 튜닝
2026-05-12,18:07:31,퇴근 (자동),
```

프로젝트명에 쉼표가 포함되면 자동으로 큰따옴표로 감싸집니다.

## 빌드

```bash
pip install pyinstaller
pyinstaller daily_reporter.spec
```

빌드 결과물은 `dist/daily_reporter.exe`에 생성됩니다.  
템플릿은 `_templates.py`에 base64로 내장되어 있으며, 빌드 시 별도 파일 없이 EXE 단독으로 배포 가능합니다.
