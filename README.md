# Daily Reporter

업무 시간을 기록하는 Windows 데스크탑 앱입니다.  
출근/퇴근을 기록하고, 매 정시마다 업무 내용을 입력할 수 있습니다.

---

## 기능

- **출근/퇴근 기록** — 클릭 한 번으로 타임스탬프와 함께 로그 파일에 저장
- **실시간 시계** — 현재 시각을 메인 화면에 표시
- **정시 알림** — 매 정시마다 알림음과 함께 업무 내용 입력 팝업 표시
- **마지막 기록 표시** — 가장 최근에 저장한 내용을 메인 화면에서 확인 가능
- **날짜별 로그 파일** — 실행 위치에 `YYYY-MM-DD.txt` 형식으로 자동 저장

---

## 사용 방법

1. `daily_reporter.exe` 실행
2. **출근** 버튼 클릭 → 출근 시각 기록 + 정시 알림 시작
3. 매 정시마다 팝업이 뜨면 업무 내용 입력 후 **기록** 버튼 또는 `Enter`
4. **퇴근** 버튼 클릭 → 퇴근 시각 기록 + 알림 종료

---

## 로그 파일 형식

실행 위치에 날짜별로 생성됩니다.

```
2026-04-11 09:00:00 출근
2026-04-11 10:00:00 기획 회의 참석
2026-04-11 11:00:00 API 개발
2026-04-11 18:00:00 퇴근
```

## 빌드

```bash
pip install pyinstaller
pyinstaller --noconsole --onefile --icon=icon.ico --add-data "icon.ico;." daily_reporter.py
```

빌드 결과물은 `dist/daily_reporter.exe`에 생성됩니다.
